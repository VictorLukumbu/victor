Victor Milamulo Lukumbu — Personal Football Portfolio

This is the official website for Victor Milamulo Lukumbu, a rising football talent from Beni, DRC. 
The site showcases Victor’s journey, achievements, and aspirations using HTML, CSS, and JavaScript.

------------------------------
Project Structure:

- index.html       → Homepage
- about.html       → Victor’s biography and career timeline
- blog.html        → Match previews and football insights
- contact.html     → Contact form and social links
- gallery.html     →

Folders:
- css/blog.css     → Global styles and theme switcher
- js/script.js     → Theme toggle logic
- assets/images/   → Photos and visuals

------------------------------
Theme Switcher Feature:

- A button with id="themeToggle" toggles light/dark mode
- JavaScript adds/removes the class "dark-mode" on <body>
- CSS styles respond to the presence of .dark-mode
